# React testing workshop starter

Minimal React 18 project for Project Web on unit, functional, and E2E testing.

## Setup

```bash
git clone https://github.com/sofiasribeiro/PW-testing.git
cd PW-testing
npm install
npm run dev
npm test
```

## Branches

- `main`: starter code with `LoginForm` + `validateLogin`, Bootstrap styling, and test scaffolding (TODOs only)
- `solution`: reference implementation with all tests fully implemented

## Student workflow (`main`)

1. Checkout the starter branch:

   ```bash
   git checkout main
   ```

2. Run the app and confirm Bootstrap is loaded:

   ```bash
   npm run dev
   ```

3. Implement the tests by editing these files:
   - `src/__tests__/validateLogin.test.js`
   - `src/__tests__/LoginForm.test.jsx`

## Stuck?

Run:

```bash
git checkout solution
```
