# React testing workshop starter

Minimal React 18 project for a 2-hour workshop on unit, functional, and E2E testing.

## Setup

```bash
git clone https://github.com/sofiasribeiro/PW-testing.git
cd PW-testing
npm install
npm test
```

## Branches

- `main`: starter code with `LoginForm` + `validateLogin` and test scaffolding (TODOs only)
- `solution`: reference implementation with all tests fully implemented

## Stuck?

Run:

```bash
git checkout solution
```
